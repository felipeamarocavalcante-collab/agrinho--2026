# agrinho--2026
projeto agrinho
link do projeto: https://felipeamarocavalcante-collab.github.io/agrinho--2026/
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”. 


Tema: Inovação verde para o campo


Objetivo: O principal objetivo da inovação verde no campo é transformar a produção agrícola e agropecuária em um modelo totalmente sustentável, aliando o aumento da produtividade à preservação ambiental e à responsabilidade social.
